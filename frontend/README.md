# AI Module for Documentation

AI-powered document management and analysis system built using React + Node.js.

## Features

- Upload documents
- View documents
- Edit documents
- AI chatbot support
- Document search
- Admin dashboard
- User dashboard
- File preview support
- Semantic search
- Analytics module

## Tech Stack

Frontend:
- React
- Vite
- CSS

Backend:
- Node.js
- Express

AI:
- Embeddings
- Semantic Search
- Chunk Processing

## Project Structure

AI_module_for_documentation/

backend/
frontend/

## Installation

Clone project

```bash
git clone https://github.com/Sruthi0921/AI_module_for_documentation.git
```

Backend:

```bash
cd backend
npm install
npm start
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```
