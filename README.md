# AI Module for Documentation

AI-powered document management system with document upload, viewing, editing, semantic search and chatbot support.

## Features

- Upload documents
- View documents
- Edit documents
- Admin dashboard
- User dashboard
- AI chatbot
- Semantic search
- Analytics

## Tech Stack

Frontend:
- React
- Vite

Backend:
- Node.js
- Express

## Run Project

Backend:

```bash
cd backend
npm install
npm start
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```
